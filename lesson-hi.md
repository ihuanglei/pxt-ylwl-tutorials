## 第一步 @fullscreen

请小朋友们拖出积木块 ``||basic:show string||`` 放到 ``||basic:forever||`` 中

```blocks
basic.forever(function () {
    basic.showString("hello")
})
```

## 第二步 @fullscreen

请小朋友们双击 hello ，然后输入 hi ，请记住能不能输入中文哦!

```blocks
basic.forever(function () {
    basic.showString("hi")
})
```
